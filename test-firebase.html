What to do if tests fail:

  1. Make sure you've created a Firebase project at Firebase Console
  2. Update js/firebase-config.js with your actual Firebase configuration
  3. Enable Authentication (Email/Password) in Firebase Console
  4. Enable Firestore Database in test mode
  5. Enable Cloud Storage in test mode
  6. Refresh this page to test again
